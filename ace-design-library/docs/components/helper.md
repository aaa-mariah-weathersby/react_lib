### Props
| Name    | Type     | Default | Description                                                                                |   |
|---------|----------|---------|--------------------------------------------------------------------------------------------|---|
| `active`  | boolean   |         | Flag to render helper. |   |
| `content`   | string   |   | Content displayed in helper chat bubble.         |   |

!!! note ""
    Any other properties supplied will be spread to the root element (native element).
